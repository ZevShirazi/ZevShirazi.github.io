Academic website generated for Zev Shirazi.
Unzip and open the folder, then use GitHub Desktop to commit and push to a repository named `zevshirazi.github.io`
or use Git to push to a GitHub Pages-enabled repo.

Quick local preview:
- Install Ruby and Bundler (see Jekyll docs) then run:
  bundle install
  bundle exec jekyll serve --livereload

This site uses the Minimal Mistakes remote theme.
